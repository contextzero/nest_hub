# Business — NEST

[English](./README.md) | [Español](./README-ES.md) | [中文](./README-ZH.md) | [Deutsch](./README-DE.md) | [Português](./README-PT.md) | **Français**

Documentation stratégique pour les fondateurs et les dirigeants qui construisent leur main-d’œuvre IA.

---

## Synthèse exécutive

NEST transforme le déploiement des agents IA dans les organisations—passant d’outils isolés à une main-d’œuvre coordonnée sur votre infrastructure, visible sur votre téléphone, régie par vos politiques.

Ce document fournit le contexte stratégique pour évaluer et adopter NEST.

---

## Pourquoi NEST ? — Analyse par premiers principes

### Le problème (état actuel)

| Partie prenante | Point de douleur | Cause profonde |
|-------------|------------|------------|
| **Fondateur/Admin** | Aucune visibilité sur l’activité IA | Outils en silos |
| **Responsable IT** | Lacunes sécurité et conformité | Absence de piste d’audit |
| **Chef d’équipe** | Impossible de superviser les agents distribués | Pas de contrôle mobile |
| **Collaborateur** | Contexte perdu entre les sessions | Mémoire non persistante |
| **ONG / État** | Pas de piste d’audit pour rapports donateurs/citoyens | Processus manuels, pas de visibilité conformité |

### La solution (valeur NEST)

| Partie prenante | Bénéfice principal | Pourquoi c’est important |
|-------------|-----------------|----------------|
| **Fondateur/Admin** | Contrôle total sans dépendance fournisseur | Votre infrastructure, vos clés, vos données |
| **Collaborateur** | Travailler depuis téléphone/tablette à la voix | Opérations réellement mobile-first |
| **Entreprise** | Un seul hub pour tous les agents | Supprime la fragmentation des outils |
| **ONG / État** | Audit prêt conformité + opérations terrain mobiles | Rapports donateurs, services aux citoyens, contexte personnel rotatif |

## Les quatre piliers

### 1. Hub pour l’entreprise

Une app remplace Slack + Notion + Trello + WhatsApp. Projets → Collaborateurs → Sessions.

- Gestion centralisée de tous les agents IA
- Organisation par projets avec accès par rôles
- Visibilité complète et piste d’audit
- Self-hosted : votre infrastructure, vos données

### 2. Mobile pour le collaborateur

PWA sur n’importe quel téléphone. Approuver depuis le bus. Pas besoin d’ordinateur portable.

- Opérations mobiles complètes via Progressive Web App
- Approbations en temps réel et contrôle des sessions
- Interface vocale (ElevenLabs) pour une saisie plus rapide
- Fonctionne en 4G ou Wi-Fi — opérations à distance réelles

### 3. Memory Bank (Souls)

Le Hub apprend chaque collaborateur. Plus besoin de réexpliquer le contexte. L’intelligence s’accumule.

- Les profils conservent préférences et habitudes de travail
- La mémoire projet retient décisions, glossaire, contexte
- Colony memory découvre les workflows optimaux
- La connaissance ne quitte pas votre infrastructure

### 4. Agent Swarm

Plusieurs agents IA spécialisés en parallèle.

- Claude Code, Codex, Cursor, Gemini, OpenCode, KiloCode
- ZeroClaw : automatisation headless avec auto-correction (**enveloppe dans `annie computer` à partir du 1er juin 2026**)
- OpenClaw : orchestration visuelle des workflows (**enveloppe dans `annie computer` à partir du 1er juin 2026**)
- Hermes : enveloppe d'usage ordinateur aux côtés d'OpenClaw et ZeroClaw (**enveloppe dans `annie computer` à partir du 1er juin 2026**)
- Darwin Agents : travailleurs IA qui évoluent avec votre organisation

## Différenciation concurrentielle

| Concurrent | Faiblesse | Avantage NEST |
|------------|----------------|-----------------|
| **Zapier** | Automatisation par règles | Natif IA, apprend, évolue |
| **Salesforce** | Orienté ventes | Chaque collaborateur, pas seulement la vente |
| **UiPath** | RPA par règles | Natif IA, pas basé sur des règles |
| **GitHub Copilot** | Code uniquement | Main-d’œuvre complète, pas seulement le code |
| **Agentforce** | SaaS uniquement | Option self-hosted, contrôle total |

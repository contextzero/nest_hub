# Positioning — NEST

## Market Position

**NEST** is positioned as the **self-hosted enterprise workforce platform** for AI agent management.

---

## Target Audience

### Primary

| Segment | Description |
|---------|-------------|
| **CTO / IT Leaders** | Want full control of AI infrastructure, no SaaS dependencies |
| **Dev Team Leads** | Need visibility into agent sessions, approval workflows |
| **Engineering Managers** | Multi-agent orchestration, audit trails |
| **Security / Compliance** | Data residency requirements, full audit capability |

### Secondary

| Segment | Description |
|---------|-------------|
| **Individual Developers** | Personal AI coding assistant with mobile control |
| **Small Teams** | 2-10 developers needing shared agent infrastructure |
| **Agencies** | Multiple clients, project-based AI work |
| **Entrepreneurs** | Solo founders managing AI agents from phone |
| **NGOs** | Field coordination, donor reporting, rotating staff context |
| **Governments** | Citizen service automation, compliance audit trails, field inspections |

---

## Competitive Landscape

### Competitors

| Competitor | Weakness | NEST Advantage |
|------------|----------|----------------|
| **GitHub Copilot** | SaaS-only, no self-hosted | Full self-hosted option |
| **Cursor** | Individual IDE, no mobile control | Mobile PWA, multi-session |
| **Anthropic Console** | SaaS, limited enterprise | Enterprise deployment, BYOK |
| **OpenAI Agents** | Beta, SaaS | Production-ready, self-hosted |
| **Amazon Q** | AWS lock-in | Cloud-agnostic |

---

## Differentiation (The Zags)

1. **Self-Hosted First** — No recurring SaaS fees, complete data ownership
2. **Mobile-First Control** — Phone/tablet PWA for real-time oversight
3. **Multi-Agent Hub** — Claude, Codex, Cursor, Gemini, OpenCode, KiloCode, ZeroClaw, OpenClaw in one platform
4. **BYOK Everything** — Your LLMs, your voice, your infrastructure
5. **Enterprise Audit** — Full Postgres audit trail, compliance-ready
6. **Zero Lock-In** — Docker + npm, no proprietary stack

---

## Value Proposition

### For CTOs

> "Run AI agents on your infrastructure with full visibility and zero SaaS dependencies."

### For Team Leads

> "Every agent session visible on your phone. Approve or reject in real time."

### For Developers

> "Code with Claude, control from your phone. Your laptop runs the agent, your tablet runs the oversight."

---

## Messaging Framework

### Problem

- AI agents are hard to control
- SaaS tools don't give infrastructure control
- No mobile visibility into agent sessions
- Vendor lock-in with AI providers

### Solution

- Self-hosted Docker deployment
- Mobile PWA for real-time control
- Multi-agent support from one hub
- BYOK: bring your own LLMs, voice, infrastructure

### Proof Points

- Docker Compose in 5 minutes
- Public Docker images (no build from source)
- CLI via npm (`npm install -g @contextzero/nest`)
- Full audit log in PostgreSQL
- Real-time SSE streaming to mobile

---

## Market Timing

The market is shifting from:
- **SaaS-only AI tools** → Self-hosted infrastructure
- **Individual agents** → Multi-agent orchestration
- **Desktop-only** → Mobile-first oversight
- **Proprietary stacks** → Open standards (Docker, npm)

NEST rides all four trends simultaneously.
